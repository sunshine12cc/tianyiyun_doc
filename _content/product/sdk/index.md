# SDK 文档

青云为开发者提供软件开发包 SDK (Software Development Kit)， 可以利用它对青云的资源进行查看、创建和操作，借此开发出更富创造力的产品。

警告

SDK 的使用受 API 访问配额的限制，具体请查看 [_概述_](../api/common/overview.html#common-overview) 中的 “API 请求配额限制”
