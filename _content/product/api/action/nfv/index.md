# nfv[](#api-nfv "永久链接至标题")

网络功能虚拟化，可以创建、删除、修改、查看一个网络组件(NFV)。网络组件用于将多个私有网络中的主机连接到外网；它可以加入多个私有网络，绑定多个公网 IP ；各个不同的私有网络可以绑定不同的路由表和防火墙。
