# 资源监控[¶](#api-monitor "永久链接至标题")

获取青云资源的监控数据，可设定时间范围及时间间隔， 目前支持对主机、硬盘、网络、路由器、负载均衡器的监控。

为减少数据传输，在保持数据结构清晰的前提下，我们对监控数据做了压缩处理。

*   [GetCacheMonitor](get_cache_monitor.html)
