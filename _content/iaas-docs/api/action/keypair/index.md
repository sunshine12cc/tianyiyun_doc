# SSH 密钥[¶](#ssh "永久链接至标题")

该功能仅对 Linux 主机有效。青云官方提供的 Linux 映像是不允许使用密码来 SSH 登录的， 只接受 SSH 公钥/私钥验证身份。您可以新建一个 SSH 密钥，并立刻下载其私钥。 请保管好私钥，因为青云是不保存您的私钥的。
